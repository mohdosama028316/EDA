# EDA
🚀 **Project Showcase: Insurance Cost Prediction – Data Preprocessing & Exploratory Data Analysis**

I'm excited to share one of my recent Data Analytics projects focused on understanding the factors that influence **medical insurance charges** through data preprocessing, feature engineering, and statistical analysis.

### 📌 Project Highlights

✅ Performed comprehensive **Exploratory Data Analysis (EDA)** to uncover patterns and trends in the dataset.

✅ Cleaned and preprocessed the data by:

* Removing duplicate records
* Handling categorical variables
* Standardizing numerical features
* Preparing the dataset for analysis

✅ Applied **Feature Engineering** by:

* Creating BMI categories (Underweight, Normal, Overweight, Obese)
* One-Hot Encoding categorical variables
* Scaling numerical features using StandardScaler

✅ Conducted **Feature Selection** using:

* Pearson Correlation Analysis
* Chi-Square Test for categorical variables

### 📊 Key Insights

* Smoking status showed one of the strongest relationships with insurance charges.
* BMI and age significantly impacted medical expenses.
* Statistical feature selection helped identify the most influential variables for future predictive modeling.
